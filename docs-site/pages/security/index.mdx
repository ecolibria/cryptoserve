## Security Highlights

### Cryptographic Primitives

| Purpose | Algorithm | Standard |
|---------|-----------|----------|
| Encryption | AES-256-GCM | NIST SP 800-38D |
| Key Derivation | HKDF-SHA256 | RFC 5869 |
| Signatures | Ed25519 | RFC 8032 |
| Key Encapsulation | ML-KEM-768 | FIPS 203 |
| Post-Quantum Signatures | ML-DSA-65 | FIPS 204 |

### Key Security Features

- **Key Commitment**: Protection against invisible salamanders attack
- **Domain Separation**: Keys cannot be reused across contexts
- **Constant-Time Operations**: No timing side channels
- **Secure Memory**: Sensitive data zeroized after use
- **FIPS 140-2/3**: Compliance mode available

### Audit Logging

```json
{

```
