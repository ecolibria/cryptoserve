## Quick Comparison

<Tabs items={['Python', 'TypeScript']}>
  <Tabs.Tab>
```python copy
from cryptoserve import CryptoServe

crypto = CryptoServe(app_name="my-app", team="platform")

# Encrypt
encrypted = crypto.encrypt_string("data", context="user-pii")

# Decrypt
decrypted = crypto.decrypt_string(encrypted, context="user-pii")
```
  </Tabs.Tab>
  <Tabs.Tab>
```typescript copy
import { crypto } from 'cryptoserve';

// Encrypt
const encrypted = await crypto.encryptString("data", { context: "user-pii" });

// Decrypt
const decrypted = await crypto.decryptString(encrypted);
```

## Installation

SDKs are personalized with your identity embedded. Get your install command from the dashboard.

```bash copy
# Python
pip install http://localhost:8001/sdk/download/YOUR_TOKEN/python

# TypeScript/JavaScript
npm install http://localhost:8001/sdk/download/YOUR_TOKEN/npm
```

## Features

- **Zero Configuration**: Identity embedded, no API keys needed
- **Automatic Algorithm Selection**: Context determines the right algorithm
- **Full Type Safety**: TypeScript types, Python type hints
- **Async Support**: Async/await in both languages
- **Framework Integrations**: Django, FastAPI, Express, Next.js
