# Quick Start

Get up and running with CryptoServe in 5 minutes.

## Prerequisites

- Docker and Docker Compose installed
- GitHub account for authentication

---

<Steps>
### Start the Services

Clone the repository and start the services:

```bash copy
git clone https://github.com/keytum/crypto-serve.git
cd crypto-serve
docker compose up -d
```

This starts:
- **Backend API** on port 8001
- **Frontend Dashboard** on port 3001
- **PostgreSQL** database

### Sign In to the Dashboard

Open [http://localhost:3001](http://localhost:3001) and click **Sign in with GitHub**.

<Callout>
  The first time you sign in, you'll need to authorize the OAuth application.
</Callout>

### Create an Identity

1. Click **Create New Identity**
2. Fill in the details:

| Field | Example | Description |
|-------|---------|-------------|
| Name | `my-backend-service` | Unique identifier |
| Team | `engineering` | Team ownership |
| Environment | `development` | Deployment environment |
| Contexts | `user-pii`, `general` | Allowed encryption contexts |

3. Click **Create Identity**

### Install the SDK

Copy the install command from the dashboard:

```bash copy
pip install http://localhost:8001/sdk/download/YOUR_TOKEN/python
```

<Callout type="warning">
  Replace `YOUR_TOKEN` with the actual token shown in your dashboard.
</Callout>

### Encrypt Your First Data

Create a Python file and try it out:

```python filename="demo.py" copy
from cryptoserve import CryptoServe

crypto = CryptoServe(app_name="my-service", team="engineering")

# Encrypt a string
encrypted = crypto.encrypt_string(
    "user@example.com",
    context="user-pii"
)
print(f"Encrypted: {encrypted[:50]}...")

# Decrypt it back
decrypted = crypto.decrypt_string(encrypted, context="user-pii")
print(f"Decrypted: {decrypted}")

# Works with binary data too
binary_data = b"sensitive binary data"
ciphertext = crypto.encrypt(binary_data, context="general")
plaintext = crypto.decrypt(ciphertext, context="general")
print(f"Binary roundtrip: {plaintext == binary_data}")
```

Run it:

```bash
python demo.py
```

Output:
```
Encrypted: eyJhbGciOiJkaXIiLCJlbmMiOiJBMjU2R0NNIiwiY...
Decrypted: user@example.com
Binary roundtrip: True
```

### View the Audit Log

Go to [http://localhost:3001/audit](http://localhost:3001/audit) to see your operations logged with:

- Timestamp
- Identity
- Operation (encrypt/decrypt)
- Context
- Algorithm used

</Steps>
